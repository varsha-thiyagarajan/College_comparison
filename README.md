# 🎓 College Comparison & Recommendation Platform

A smart web platform that helps students compare colleges and make better admission decisions using data-driven insights like fees, placements, ROI, and personalized recommendations.

---

## 🚀 Problem Statement

Students often face difficulty choosing the right college due to scattered, incomplete, or confusing information.

This platform solves that problem by:

* Centralizing key college data
* Providing side-by-side comparison
* Offering intelligent, personalized recommendations

---

## ✨ Features

### 🔍 Smart Search & Filtering

* Search colleges by:

  * Course
  * Location
  * Budget
* Rank-based categorization:

  * 🟢 Safe
  * 🟡 Match
  * 🔴 Dream

---

### ⚖️ College Comparison

* Compare up to **3 colleges simultaneously**
* Compare key factors:

  * Fees
  * Location
  * Rating
  * Placement stats
  * ROI

---

### 📊 ROI Analyzer (Highlight Feature)

* Calculates **Return on Investment**
* Visual comparison using dynamic bars
* Automatically highlights the **best ROI college**

---

### 🏫 Deep College Profiles

Each college includes:

* 💰 Fees breakdown (tuition, hostel, other)
* 📈 Placement statistics (average, highest, median)
* 🏫 Infrastructure details
* 👨‍🏫 Faculty quality insights
* 🏅 Accreditation (NAAC, NBA)

---

### 🤖 Smart Recommendation Engine

Recommends top colleges based on:

* Rank
* Budget
* Course
* User preference:

  * High placement
  * Low fees
  * Balanced

Includes scoring-based ranking system.

---

### 🎯 Admission Predictor

Predicts admission chances using rank:

* 🟢 Safe colleges
* 🟡 Match colleges
* 🔴 Dream colleges

---

### 🔄 Smart Selection UX

* Toggle Select / Deselect
* Visual highlighting of selected colleges
* Limit of 3 selections for comparison

## ⚙️ How to Run Locally

### 1️⃣ Clone the repository

```
git clone https://github.com/YOUR_USERNAME/college-comparison-platform.git
```

### 2️⃣ Navigate to backend

```
cd server
```

### 3️⃣ Install dependencies

```
npm install
```

### 4️⃣ Start server

```
node server.js
```

### 5️⃣ Open frontend

Open:

```
client/index.html
```

in your browser
